# Altus Logo Files Created

This directory contains local logo files for the Altus Loan Management System to ensure reliable logo display even when external resources are unavailable.

## Files:
- `altus-logo.svg` - Primary vector logo (scalable, small file size)
- `altus-logo.png` - Fallback raster logo (for better browser compatibility)

## Logo Fallback System:
1. **Local SVG** - Primary choice (altus-logo.svg)
2. **External PNG** - Secondary fallback (altusfinancialservices.com)
3. **Text Logo** - Final fallback (always works)

The logo component automatically tries each option in order if the previous one fails to load, ensuring the logo always displays correctly regardless of network issues or external site availability.
